Política de Privacidade

Kiddo — Criando hábitos, formando vencedores

Esta Política de Privacidade explica como o Kiddo trata informações quando você usa o app. O Kiddo foi pensado para funcionar principalmente com dados armazenados no próprio dispositivo.

Dados que o app pode armazenar

  • Informações da conta (nome, e-mail/usuário e avatar).
  • Progresso diário (tarefas marcadas), XP e sequência (ofensiva).
  • Hábitos personalizados criados pelo usuário.

Onde os dados ficam

Os dados ficam armazenados localmente no dispositivo por meio do armazenamento do WebView (LocalStorage). Isso permite que o app funcione mesmo sem internet.

Login com Google

Se você optar por entrar com Google, o app utiliza o Google Sign-In para obter seus dados básicos de perfil (como nome, e-mail e foto). Esses dados são usados para criar/identificar sua conta dentro do app.

Notificações

O app pode enviar lembretes de notificação. Você pode controlar as permissões de notificação nas configurações do Android.

Compartilhamento

O Kiddo não vende dados pessoais. O app não possui um servidor próprio para receber o seu progresso.

Exportação de dados

O app possui uma função para exportar seus dados para um arquivo JSON, para backup.

Segurança

Nenhum sistema é 100% seguro. Recomendamos proteger o seu dispositivo com senha/biometria.

Contato

Para dúvidas sobre privacidade, entre em contato pelos canais informados no app.

Mudanças nesta política

Podemos atualizar esta política para refletir melhorias no app. Recomendamos revisar periodicamente.
